Justify-content The justify-content property in Flexbox is used to control the alignment of flex items along the main axis of the flex container. It determines how the extra space or insufficient space along the main axis is distributed.

question 2

Ans:

In a flexbox container, the flexbox items are aligned on the cross axis, which is vertical by default (opposite of flex-direction).

Ans:

The gap property defines the size of the gap between the rows and between the columns in flexbox

Ans:

The flex-direction property specifies the direction of the flexible items.

question 5

Ans:

The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
